How to Play

Color each shape in the shape map. Don't let any two shapes that share an edge have the same color!

A random color is available for selection for second. In the color box, the currently active color is shown on the left. The smaller color on the right is a preview of the next color.

Score is determined by the current level and time to completion. Points are subtracted if a level is timed out before it is completed.

NB: It is ok for two shapes that share a single point to be the same color.
